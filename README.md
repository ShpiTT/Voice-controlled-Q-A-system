# 人机语音交互系统

## 项目简介
基于百度语音API的人机交互系统，通过语音指令控制**电脑程序**和**手机设备**运行，并将结果通过语音合成播报。

## 功能特性
- ✅ 语音指令输入（ASR语音识别）
- ✅ 控制电脑程序运行
- ✅ **ADB控制手机设备**（新增）
- ✅ 程序结果语音播报（TTS语音合成）

## 安装步骤

### 1. 安装Python依赖
```bash
pip install -r requirements.txt
```

### 2. 配置百度API密钥
在 `main.py` 中修改以下配置：
```python
API_KEY = '你的API_KEY'
SECRET_KEY = '你的SECRET_KEY'
```

### 3. 配置ADB（手机控制功能）
1. 下载 [Android SDK Platform Tools](https://developer.android.com/studio/releases/platform-tools)
2. 解压后将路径添加到系统环境变量 PATH
3. 手机开启USB调试模式（设置 → 开发者选项 → USB调试）
4. 用USB线连接手机和电脑
5. 在手机上允许USB调试授权

验证ADB安装：
```bash
adb devices
```

### 4. 运行程序
```bash
python main.py
```

## 使用说明
1. 运行程序后，按Enter键开始录音
2. 说出您的指令（5秒录音时间）
3. 系统会识别语音并执行相应操作
4. 执行结果将通过语音播报

## 百度语音API申请
1. 访问 [百度AI开放平台](https://ai.baidu.com/)
2. 注册/登录账号
3. 创建应用，开通"语音识别"和"语音合成"服务
4. 获取 API_KEY 和 SECRET_KEY

## 技术实现
- **语音采集**: PyAudio 实时录音
- **语音识别**: 百度语音识别API (ASR)
- **语音合成**: 百度语音合成API (TTS)
- **音频播放**: Pygame
- **系统信息**: Psutil
- **手机控制**: ADB (Android Debug Bridge)

## 系统架构
```
用户语音 → 麦克风录音 → 百度ASR识别 → 指令解析
                                          ↓
                              ┌───────────┴───────────┐
                              ↓                       ↓
                         PC程序控制              ADB手机控制
                              ↓                       ↓
                         执行结果 ←─────────────────────┘
                              ↓
                      百度TTS语音合成 → 播放结果
```

## 注意事项
- 需要麦克风设备
- 需要网络连接（调用百度API）
- Windows系统下运行
- 手机控制需要安装ADB并连接手机
- 手机需开启USB调试模式
